# MojePierwszeRepo

Repozytorium zostało utworzone w ramach ćwiczenia z obsługi platformy GitHub.

## Cel zadania

Celem zadania było zapoznanie się z podstawowymi funkcjami GitHub, takimi jak tworzenie repozytorium, dodawanie plików, tworzenie gałęzi, wykonywanie pull requestów, korzystanie z issues oraz dokumentowanie projektu.

## Zawartość repozytorium

W repozytorium znajdują się przykładowe pliki utworzone podczas ćwiczenia:

* `Witaj.txt` – pierwszy plik tekstowy dodany do repozytorium,
* `NowaFunkcja.txt` – plik dodany na gałęzi `rozwoj`,
* `Program.cs` – kod gry Saper w języku C#,
* `README.md` – dokumentacja repozytorium.

## Opis projektu

W ramach zadania przeanalizowano kod gry Saper oraz naprawiono znalezione błędy. Każda poprawka została wykonana osobno i zapisana jako oddzielny commit, aby można było prześledzić historię zmian w repozytorium.

## Wykonane czynności

* utworzenie repozytorium `MojePierwszeRepo`,
* dodanie pliku `Witaj.txt`,
* utworzenie gałęzi `rozwoj`,
* dodanie pliku `NowaFunkcja.txt`,
* utworzenie pull requesta,
* scalenie zmian z gałęzią główną,
* utworzenie issue,
* dodanie i poprawa kodu gry Saper,
* zapisanie każdej poprawki jako osobny commit.

## Uruchomienie programu

Program `Program.cs` można uruchomić w środowisku Visual Studio jako aplikację konsolową C#. Po uruchomieniu użytkownik podaje współrzędne pola na planszy Sapera, a program odkrywa wybrane pole i sprawdza, czy gracz trafił na bombę.
